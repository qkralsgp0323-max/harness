## 6. 주요 기능
메인 페이지 배너 : 배너 안에 버튼을 넣어 가이드 페이지로 연동
카테고리 및 상품 카드 : 각각의 카드를 호버했을 때 기본의 사진과 다른 사진이 보여지며 클릭 시, 각각의 페이지로 이동
메인 페이지 영상 섹션 : GSAP의 스크롤 애니메이션 기능을 사용하여 페이지 스크롤을 내릴 시 안 보이던 영상이 보여지며 영상 섹션의 영역이 100vh였다가 작아지게 만듦
리뷰 슬라이드 : 사이트 내의 베스트 리뷰들을 모아 한 줄로 나열하여 슬라이드 애니메이션을 만들어 무한동력으로 가로로 슬라이드가 됨 또한 마우스를 호버하였을 시, 슬라이드 애니메이션이 호버가 되어있는 중에는 멈춤 -> 사용자 인터페이스를 구상 후 구현함

## 7. 트러블 슈팅
문제 1. 간략한 내용
상황: 팀원 각각의 파일을 커밋하고 병합하는 과정에서 파일간의 충돌과 오류로 병합에 문제가 발생
해결: 이전의 브런치들을 삭제 및 재병합하여 해결
과정: 각자의 컴퓨터로 5명이 모여 하나의 쇼핑몰 사이트를 만드는 과정에서 메인 브런치에서 병합되어야 하는 각자의 파일들이 경로가 엉켜 각각의 파일이 병합이 안 되고, 개개인의 브런치에만 완성본이 존재하게 됨 -> 병합이 되지 않아 변경 사항이 적용이 안 됨 -> 메인에 존재하던 수정 이전의 파일들을 삭제하여 새로이 브런치를 병합해 문제를 해결하였다. 다음 프로젝트에선 이러한 상황이 발생하지 않도록 메인에 병합 요청을 한 후, 병합이 되면 존재하던 이전의 브런치들을 삭제하여 경로 이상이 생기지 않도록 주의해야할 것 같다.
